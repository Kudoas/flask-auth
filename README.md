# flask-jwt-extendを使った認証機能の実装

- signup時のpasswordのハッシュ化
- login時にjwtを発行し、セッション管理
- user_formのエラーハンドリング
- jwt tokenによるAPI endpointの保護

## library

- Flask
- Flask-JWT-extended
- hashlib

## enviroment

- python 3.8.5
- poetroy
